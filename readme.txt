	1. OPEN SERVER.
	2. IMPORT the given MYSQL database (store.sql) in the ROOT user directory.
	3. IF you are on wamp server copy the "shopping_site" folder to "C://wamp64/www/<HERE>" in {64 bit}  /  "C://wamp32/www/<HERE>" in {32 bit}.
	3. TYPE ADDRESS " localhost/shopping_site" in the browser.
	4. ENJOY.
